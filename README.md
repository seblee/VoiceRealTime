# VoiceRealTime

# 1.windows分别提供了x86, x64平台下的debug和release版本库，由于sdk是用vs2013开发的，现在不确定其它vs版本会出现什么问题, 所以请最好也使用vs2013进行相关使用。
# 2.ssl, crypto, opus都为第三方库(openssl 1.0.2j)，用户可以根据自己要求自行配置，sdk本身提供的仅作参考
# 3.目前linux平台只提供了64位版本库，同时编译时请注意编译选项，具体请查看demo文件下config文件。
# 4.在使用sdk之前，请确保已认真阅读阿里云官网相关文档。
# 5.windows平台编译完成后，需要添加对应的dll文件到执行文件目录。
# 6.demo目录下提供了一个可以识别的.wav文件
